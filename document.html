Upload

This is a function to upload the training data.

STEP 1:

Users can choose the options which you want to upload in the Select Table.
Now, we choose the option '焊點'

STEP 2:

When window shows up, users can click and drag the data to window.

STEP 3:

The progress bar will show on and it represents the progress of upload, and you can upload multiple data at the same time.

Download

This is a function to download the particular training data for PINE server.

STEP 1:

Select the table you want to download and set some attributes of it in the 'Select attributes' area.
Now, we choose the option '焊點' and all of 'Select attributes'

STEP 2:

Push 'X' of the selected attribute to delete what you don’t want.
Now, we delete the attribute 'raw'.

STEP 3:

Push 'Search' button in the 'Select attributes' area to show the data below.

STEP 4:

Add message in 'Search Data' area and push 'Search' button beside that to search particular data. Now, we input 'NG' in 'Search Data' and push 'Search' button.

STEP 5:

Push 'Download' button to download these data to your computer.

Managment

This is a function for managers to check users' permissions and change them.

STEP 1:

Search the user you want to check or change his permissions and Click the user's name to enter 'edit' page.
Now, we choose the 'User1_1@test.com'

STEP 2:

Change his permission and push 'Update' button to confirm.
