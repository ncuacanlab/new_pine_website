Managment

This function is for manager to manage users’ permission. You can access through the ‘Management’ button in the menu.

First, you can see search page. It’ll show the users’ names and their roles. You can find any user by searching him. For example, if you want to find some users in group sub3_1, you can just type sub3_1 in the ‘Search’ blank.

To add the roles of sub3_2@test.com into sub3_1, you can click it and choose sub3_1 in Roles in the page below.

Finally, chick ‘Update’ button to confirm and the user, sub3_2@test.com, can access resources in group sub3_1.
